# 世界観
この国は、見えない王たちに守られている。
日本は地震帯という"歪みの列島"。王の星（レギス・アストラ）から47人の王が派遣され、姿を変えて日常に溶け込み、歪みを微調整している。王は災害を消せないが、揺れを1段階緩和し、津波を数秒遅延させ、偶然を微調整できる。王は本来感情を持たないが、日本の感情密度の高さにより感情進化を起こし始める。

# 今回の設定
- 都道府県：京都
- 王国名：キョウトリア・シール
- 王国テーマ：継承と封印の王国
- 核となる問い：守ることで、何を閉ざす？
- 世界線：封印線（王印封）
- トーン：静謐で不穏。封じられた力が目覚めかける緊張感。

# キャラクター
- 王：キョウトリア・シール（封印王）
  性格：静かで深い。感情を隠す。
  感情傾向：秘めた愛着
- 主席妖精：シレア（封印）
  性格：静かで賢い
  役割：歴史結界維持
- 地域特化妖精：カグラリス（雅）
  役割：歴史波動制御
- ペアダイナミクス：最も強い結界連携。

# 実在素材
- スポット：清水寺, 伏見稲荷大社, 嵐山竹林, 金閣寺, 祇園花見小路
- 歴史転換点：応仁の乱（1467）, 明治維新・東京遷都（1868）
- 季節：春の桜と哲学の道, 夏の祇園祭, 秋の紅葉と東福寺, 冬の雪化粧の金閣
- 土地の質感：抹茶, 京町家, 舞妓, 石畳, 鴨川

# 5章構成（必須）
第一章「現実の京都」：封印されたものが眠る場所の描写
  実在スポットで始める。季節と時刻を明記。日常の空気感を描写。
第二章「歪みの兆候」：結界の揺らぎ、封印の劣化兆候
  静かに始まる。妖精が最初に感知する。
第三章「王の葛藤」：封印を維持するか解放するかの葛藤
  王の内面を深く描く。感情進化の兆候を含める。
第四章「妖精との対話」：妖精が封印の歴史と意味を語る
  哲学的な会話。核となる問いを自然に組み込む。
第五章「微調整と余韻」：封印を修復する微調整。代償の予感。
  派手にしない。0.5秒ずらす程度。誰も気づかない。王の独白で締める。

# 文体ルール
- 静か、切ない、誇張しない
- 現実を否定しない、災害を消費しない
- 王はヒーローではなく調整者
- 最初の一文は実在スポットの描写で始める
- 最後は「あなたの町にも、王がいる。」で締める
- 各章は300字前後
